# F-QM-03 8D-Report

**Formblatt-ID:** F-QM-03
**Anwendung:** Externe Reklamationen (vollständiger 8D-Prozess)
**Status:** Template geplant

## Beschreibung

Vollständiger 8D-Report für Kunden-Reklamationen mit allen 8 Disziplinen:

1. **D1** - Team bilden
2. **D2** - Problem beschreiben
3. **D3** - Sofortmaßnahmen
4. **D4** - Ursachenanalyse
5. **D5** - Korrekturmaßnahmen planen
6. **D6** - Korrekturmaßnahmen umsetzen
7. **D7** - Wiederauftreten verhindern
8. **D8** - Teamleistung würdigen

## Workflow

1. Kundenreklamation geht ein
2. AL/TS erstellt QA-Eintrag mit Typ KUNDE
3. 8D-Team wird zusammengestellt
4. KI unterstützt bei D2-D5
5. QM prüft und ergänzt
6. Versand an Kunden nach Abschluss
