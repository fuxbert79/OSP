# F-QM-04 NZA (Nach-/Zusatzarbeiten)

**Formblatt-ID:** F-QM-04
**Anwendung:** Interne Reklamationen
**Status:** Template geplant

## Beschreibung

Dokumentation von Nach- und Zusatzarbeiten (NZA) bei internen Fehlern in der Fertigung.

## Anwendungsfälle

- Crimpfehler
- Abisolierfehler
- Falsche Längen
- Beschädigungen
- Verwechslungen

## Dateien (geplant)

| Datei | Beschreibung |
|-------|--------------|
| `F_QM_04_template.md` | Markdown-Template |
| `F_QM_04_schema.json` | JSON-Schema für Validierung |
| `F_QM_04_prompt.md` | KI-Prompt für automatische Ausfüllung |

## Workflow

1. Fehler wird in Fertigung entdeckt
2. NZA-Meldung durch MA an Vorarbeiter
3. AL/TS erstellt QA-Eintrag mit Typ INTERN
4. Dokumentation der Nacharbeit
5. Statistik-Auswertung für KVP

## Besonderheiten

- **Schnelle Erfassung** - Minimale Pflichtfelder
- **KST-Zuordnung** - Verursacher-Kostenstelle
- **Zeiterfassung** - Aufwand für Nacharbeit
