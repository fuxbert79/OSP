# RMS Backend

**Status:** Geplant
**Technologie:** FastAPI (Python 3.11+)

## Geplante Funktionen

- REST API für Reklamationsverwaltung
- SharePoint-Anbindung
- Claude API Integration für Formblatt-Ausfüllung

## Endpunkte (geplant)

| Methode | Pfad | Beschreibung |
|---------|------|--------------|
| GET | /api/reklamationen | Liste aller Reklamationen |
| GET | /api/reklamationen/{id} | Einzelne Reklamation |
| POST | /api/reklamationen | Neue Reklamation |
| PUT | /api/reklamationen/{id} | Reklamation aktualisieren |
| POST | /api/formblatt/generate | KI-Formblatt generieren |
